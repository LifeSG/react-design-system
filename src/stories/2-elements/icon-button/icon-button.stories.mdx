# Icon Button

A call to action component with an icon in it.

## Props

The component also inherits the [HTMLButtonElement](https://developer.mozilla.org/en-US/docs/Web/API/HTMLButtonElement) attributes.

## Modifying the style

You can modify the style of the `IconButton` simply by extending it when using it with styled-components.

<StyledIconButton>
    <Icon type="cog" style={{ fontSize: "1.25rem" }} />
</StyledIconButton>

```tsx
// styles.tsx

const StyledIconButton = styled(IconButton)`
    border: 1px solid green;
`;

// component.tsx

<StyledIconButton>
    <Icon type="cog" />
</StyledIconButton>;
```

If you are using css, then you can just pass the style class to the `className` prop.

```tsx
<IconButton className="my-custom-class">
    <Icon type="home" />
</IconButton>
```
