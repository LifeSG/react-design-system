# TextList

Renders a group of related contents within a list format.

The `TextList` component comes in 2 variations. **Unordered** and **Ordered**.

## Unordered List

This list is used when order does not matter and it does not follow a numbering system for each list item.

<Preview>
    <TextList.Ul>
        <li>First</li>
        <li>Second</li>
        <li>Third</li>
    </TextList.Ul>
</Preview>

You can also choose the bullet type from the selection. Types include `disc`, `circle`, `square`, `none`.

<Preview>
    <TextList.Ul bulletType="circle">
        <li>First</li>
        <li>Second</li>
        <li>Third</li>
    </TextList.Ul>
</Preview>

## Ordered list

This list is used when order matters and it follows a numbering system for each list item.

<Preview>
    <TextList.Ol>
        <li>First</li>
        <li>Second</li>
        <li>Third</li>
    </TextList.Ol>
</Preview>

You can customize counter type and the counter separator as well. Counter types include `lower-alpha`, `decimal`, `lower-roman`.

<Preview>
    <TextList.Ol counterType="lower-alpha" counterSeparator="=">
        <li>First</li>
        <li>Second</li>
        <li>Third</li>
    </TextList.Ol>
</Preview>

## Nested list

You can stack list in a nested format.

<Preview>
    <TextList.Ol>
        <li>First</li>
        <li>Second</li>
        <li>
            Third
            <TextList.Ul>
                <li>Nested First</li>
                <li>Nested Second</li>
                <li>Nested Third</li>
            </TextList.Ul>
        </li>
        <li>Fourth</li>
    </TextList.Ol>
</Preview>

Here is an example of three layers of nesting.

<Preview>
    <TextList.Ol>
        <li>First</li>
        <li>Second</li>
        <li>
            Third
            <TextList.Ol>
                <li>Nested First</li>
                <li>
                    Nested Second
                    <TextList.Ol counterType="lower-alpha">
                        <li>Nested Second Nested First</li>
                        <li>Nested Second Nested Second</li>
                        <li>Nested Second Nested Third</li>
                    </TextList.Ol>
                </li>
                <li>Nested Third</li>
            </TextList.Ol>
        </li>
        <li>Fourth</li>
    </TextList.Ol>
</Preview>

## Custom start count (Only for OrderedLists)

<Preview>
    <TextList.Ol start={7}>
        <li>First</li>
        <li>Second</li>
        <li>Third</li>
    </TextList.Ol>
</Preview>

## Reverse count (Only for OrderedLists)

When using `reversed`, you will need to specify the `start` value.

<Preview>
    <TextList.Ol reversed start={3}>
        <li>Item A</li>
        <li>Item B</li>
        <li>Item C</li>
    </TextList.Ol>
</Preview>

## Advanced Usage

### Customising the counter

Under some circumstances, you might want to customise the counter to suit your use case.

<Preview>
    <TextList.Ol counterSeparator=".">
        <li>Item A</li>
        <li>
            Item B
            <CustomOrderedList>
                <li>Nested 1</li>
                <li>Nested 2</li>
            </CustomOrderedList>
        </li>
        <li>
            Item C
            <CustomOrderedList>
                <li>Nested 1</li>
                <li>Nested 2</li>
            </CustomOrderedList>
        </li>
        <li>Item D</li>
    </TextList.Ol>
</Preview>

You can do so by overriding the default styling.

```tsx
export const CustomOrderedList = styled(TextList.Ol)`
    margin-left: 2rem;

    li {
        margin-left: 1rem;
    }

    li:before {
        content: counters(list, ".") ".";
        left: -3rem;
    }

    ${MediaQuery.MaxWidth.mobileL} {
        li {
            margin-left: 1rem;
        }
        li:before {
            left: -3rem;
        }
    }
`;
```

## Props

#### Base Props

| Name                 |                                                               Description                                                               |   Default |
| -------------------- | :-------------------------------------------------------------------------------------------------------------------------------------: | --------: |
| **children**`*`      |                                       The content of the list <br />`JSX.Element` `JSX.Element[]`                                       |         - |
| **bottomMargin**     |                    Specifies if the list is to have a margin below (in `rem` with base pixel of `16`) <br />`number`                    |         - |
| **size**             |      The text size of the list<br />Refer [here](/docs/collections-fonts-lifesg-font-set--story-placeholder) for the various sizes      |    `Body` |
| **bulletType**       |             <em>**UnorderedList prop**</em> <br />The bullet type for unordered list <br />`disc` `circle` `square` `none`              |    `disc` |
| **counterSeparator** | <em>**OrderedList prop**</em> <br />The separator between the button and the text. It can be any string value specified <br /> `string` |         - |
| **counterType**      |           <em>**OrderedList prop**</em> <br /> The numbering for the ordered list <br />`lower-alpha` `decimal` `lower-roman`           | `decimal` |
| **start**            |                         <em>**OrderedList prop**</em> <br /> The start value of the ordered list <br />`number`                         |         - |
| **reversed**         |                <em>**OrderedList prop**</em> <br /> Specifies if list should be reversed (e.g. 5,4,3...) <br />`boolean`                |   `false` |
