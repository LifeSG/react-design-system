# Text

The component that is used for headers, labels and normal text in a website.

## Usage

The `Text` component is broken down into static text and hyperlinks.

For **static text**, it is categorized into:

-   Displays
-   Headers
-   General

You can simply use the component as such:

```
import { Text } from "react-lifesg-design-system";

<Text.Body>This is my body text</Text.Body>
```

For **hyperlinks** you can use the component as such:

```
import { Text } from "react-lifesg-design-system";

<Text.Hyperlink.Default href="www.somelink.com">This is my link</Text.Hyperlink.Default>
```

## Props

For hyperlinks, the component also inherits the
[HTMLAnchorElement](https://developer.mozilla.org/en-US/docs/Web/API/HTMLAnchorElement)
attributes.

Custom props include:

| Name        |                                                                        Description                                                                         |                                                      Default |
| ----------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------: | -----------------------------------------------------------: |
| `weight`    |                                  Specifies the weight of the component <br /> `"regular"` `"semibold"` `"bold"` `"light"`                                  | Display and Header are `"bold"`<br /> General is `"regular"` |
| `external`  | **Applies to hyperlinks only**. It is an indication if the link is external to the domain. Adds an extra indicator at the end of the link <br /> `boolean` |                                                      `false` |
| `paragraph` |              ** Applies to General text only**. Adds an extra bottom margin to non header text to allow a better separation of blocks of text              |                                                      `false` |
| `inline`    |                               Sets the text to an inline display to allow a combination of text components in a single line                                |                                                      `false` |
