# ToggleButton

A button that allows a user to enter a response by clicking, and shown with a different state when activated.

The `ToggleButton` comes in 3 states, and is controlled by the `checked` prop.

| State    |        Description        | `checked` value |
| -------- | :-----------------------: | --------------: |
| Default  | When no selection is made |     `undefined` |
| Active   |      When you select      |          `true` |
| Inactive |     When you unselect     |         `false` |

## Usage

The `ToggleButton` can be used as a checkbox or a radio button. In the case of the radio button, once it has been selected, the user is
unable to unselect it. By default, it works as a checkbox.

The behaviour can be controlled by the `type` prop.

<Preview>
    <Container>
        <li>
            <ToggleButton type="radio">Radio button</ToggleButton>
        </li>

</Preview>

### Event Handling

While the component inherits its props from the [HTMLInputElement](https://developer.mozilla.org/en-US/docs/Web/API/HTMLInputElement), we recommend using the `onChange` event handler to track changes in the `ToggleButton`

    </Container>
</Preview>

## Props

The component also inherits the [HTMLInputElement](https://developer.mozilla.org/en-US/docs/Web/API/HTMLInputElement) attributes.

Custom props for this component are as such:

| Name            |                          Description                           |      Default |
| --------------- | :------------------------------------------------------------: | -----------: |
| **children**`*` |           The content of the button <br /> `string`            |            - |
| **type**        | The type of the **ToggleButton** <br /> `"checkbox"` `"radio"` | `"checkbox"` |
