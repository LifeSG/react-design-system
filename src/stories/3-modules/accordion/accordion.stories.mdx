## Constructing the Accordion

The Accordion is made up of 2 supporting components.

-   `Base` - It serves as the outer container of the Accordion, holding the title as well as a collapse/expand all mechanism
-   `Item` - This is each of the element in the Accordion. Users are free to construct any content of their choice.

<Preview>
    <Accordion.Base title="This is the Base">
        <Accordion.Item title="This is Item">
            <Text.Body paragraph>
                Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do
                eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut
                enim ad minim veniam, quis nostrud exercitation ullamco laboris
                nisi ut aliquip ex ea commodo consequat.
            </Text.Body>
            <TextList.Ul>
                <li>The first item</li>
                <li>The second item</li>
                <li>The third item</li>
            </TextList.Ul>
        </Accordion.Item>
    </Accordion.Base>
</Preview>

## Props

| Name                  |                                      Description                                       | Default |
| --------------------- | :------------------------------------------------------------------------------------: | ------: |
| **children**`*`       |          The content of the **Accordion**<br /> `JSX.Element` `JSX.Element[]`          |       - |
| **title**             |              The label text for the **Accordion** header <br /> `string`               |       - |
| **enableExpandAll**   | Specifies if the feature to expand/collapse all child items is enabled<br /> `boolean` |  `true` |
| **showTitleInMobile** |       Specifies if the title should be shown in mobile viewports<br /> `boolean`       | `false` |
| **className**         |                    Class selector for the component<br /> `string`                     |       - |

### Accordion Item Props

| Name            |                                Description                                |   Default |
| --------------- | :-----------------------------------------------------------------------: | --------: |
| **children**`*` | The content of the **Accordion.Item**<br /> `JSX.Element` `JSX.Element[]` |         - |
| **title**`*`    |     The label text for the **Accordion.Item** header <br /> `string`      |         - |
| **expanded**    |             Specifies if the item is expanded<br /> `boolean`             |         - |
| **data-testid** |               The test id for the component<br /> `string`                |         - |
| **type**        |       Sets the style type of the component<br /> `default` `small`        | `default` |
| **className**   |              Class selector for the component<br /> `string`              |         - |
