# AlertBox

A component that provides contextual feedback for certain user actions or critical information.

#### Variants

<Canvas>
    <Story name="AlertBox">
        <>
            <AlertBox.Base type="error">
                This will notify users of an error that has occurred.
            </AlertBox.Base>
            <br />
            <AlertBox.Base type="warning">
                This will serves as a warning or disclaimer to users.
            </AlertBox.Base>
            <br />
            <AlertBox.Base type="success">
                This will serves as a acknowledgement of a successful action.
            </AlertBox.Base>
        </>
    </Story>
</Canvas>

#### With action link

You can also add an action link to the alert box to allow the user to perform a further action.

<Preview>
    <AlertBox.Base
        type="warning"
        actionLink={{
            children: "Click here",
            href: "https://www.google.com",
            target: "_blank",
            rel: "noopener noreferrer",
        }}
    >
        This warning alert comes with an action link
    </AlertBox.Base>
</Preview>

#### Advanced usage

If you require text of different font weights, then you can use the `AlertBox.Description` to do so.

<Preview>
    <AlertBox.Base
        type="warning"
        actionLink={{
            children: "Click to acknowledge",
            href: "https://www.google.com",
            target: "_blank",
            rel: "noopener noreferrer",
        }}
    >
        <AlertBox.Description>
            You can add text of a{" "}
            <AlertBox.Description weight="bold">
                different weight
            </AlertBox.Description>{" "}
            to signify or highlight certain information.
        </AlertBox.Description>
    </AlertBox.Base>
</Preview>

## Props

| Name            |                                                                            Description                                                                             |   Default |
| --------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------: | --------: |
| **children**`*` |                                                    The content of the **AlertBox**<br /> `string` `JSX.Element`                                                    |         - |
| **type**`*`     |                                                   The style type of the alert<br /> `success` `warning` `error`                                                    | `success` |
| **actionLink**  | The attributes of an action link that performs an action on click<br /> [HTMLAnchorAttributes](https://developer.mozilla.org/en-US/docs/Web/API/HTMLAnchorElement) |         - |
| **className**   |                                                   The style class to override the default styling<br /> `string`                                                   |         - |
| **data-testid** |                                                               The id used for testing<br /> `string`                                                               |         - |
