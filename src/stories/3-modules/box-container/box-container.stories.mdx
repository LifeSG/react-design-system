# BoxContainer

Display contents in a box styling layout that can be collapsible if specified.

#### Default (Collapsible)

<Canvas>
    <Story name="BoxContainer">
        <BoxContainer title="This is the title">
            <div style={{ padding: "2rem" }}>
                <Text.Body>
                    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed
                    do eiusmod tempor incididunt ut labore et dolore magna
                    aliqua. Ut enim ad minim veniam, quis nostrud exercitation
                    ullamco laboris nisi ut aliquip ex ea commodo consequat.
                </Text.Body>
            </div>
        </BoxContainer>
    </Story>
</Canvas>

#### With a display state

Specifying the display state allows you to render an icon alongside the title. It is good to indicate if the contents
of the `BoxContainer` has some special attention required.

There are currently 3 states:

-   `default` - where no icon will be rendered
-   `error` - a red icon
-   `warning` - an orange icon

In this example, we are using the `error` display state.

<Preview>
    <BoxContainer title="This comes with an error icon" displayState="error">
        <div style={{ padding: "2rem" }}>
            <Text.Body>
                Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do

#### Specifying a custom call to action

You can also specify a custom call to action that will appear on the header of the `BoxContainer`.

<Preview>
    <BoxContainer
        title="This has a call to action button"
        expanded={true}
        callToActionComponent={
            <Button.Default styleType="light">Edit</Button.Default>
        }
    >
        <div style={{ padding: "2rem" }}>
            <Text.Body>
                Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do
                eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut
                enim ad minim veniam, quis nostrud exercitation ullamco laboris
                nisi ut aliquip ex ea commodo consequat.
            </Text.Body>
        </div>
    </BoxContainer>
</Preview>

## Props

| Name                      |                                         Description                                         |                   Default |
| ------------------------- | :-----------------------------------------------------------------------------------------: | ------------------------: |
| **title**`*`              |               The label text for the **BoxContainer** header <br /> `string`                |                         - |
| **children**`*`           |           The content of the **BoxContainer**<br /> `JSX.Element` `JSX.Element[]`           |                         - |
| **collapsible**           |           Specifies if the contents can be collapsed or expanded<br /> `boolean`            |                    `true` |
| **expanded**              | Specifies if the contents are expanded. Only valid if `collapsible` is true<br /> `boolean` |                   `false` |
| **callToActionComponent** |            A custom call to action component for the header<br /> `JSX.Element`             |                   `false` |
| **displayState**          |     Specifies the display state which renders an icon<br /> `default` `error` `warning`     |                 `default` |
| **className**             |               The style class to override the default styling<br /> `string`                |                         - |
| **data-testid**           |                           The id used for testing<br /> `string`                            |                         - |
| **subComponentTestIds**   |             The ids used for testing the label and handle button<br /> `string`             | `{label: "", handle: ""}` |
