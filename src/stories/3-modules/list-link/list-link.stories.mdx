## Props

#### LinkList

| Name            |                                        Description                                         |   Default |
| --------------- | :----------------------------------------------------------------------------------------: | --------: |
| **items**`*`    |     The list items to be shown <br /> `ListLinkItemProps<Type>[]` (see details below)      |         - |
| **maxShown**    |    The maximum number of items to be shown, while the rest are minimized<br /> `number`    |         - |
| **style**       |                 The size style for the `LinkList`<br /> `default` `small`                  | `default` |
| **onItemClick** | The click handler when the item is clicked<br /> `(item: LinkListItemProps<Type>) => void` |         - |
| **className**   |               The style class to override the default styling<br /> `string`               |         - |
| **data-testid** |                           The id used for testing<br /> `string`                           |         - |

#### LinkListItemProps

This also inherits the [HTMLAnchor](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a) properties.

> Do note that specifying the `onClick` prop will override the `onItemClick` event handler

| Name            |                        Description                         | Default |
| --------------- | :--------------------------------------------------------: | ------: |
| **title**`*`    |           The title to be shown <br /> `string`            |       - |
| **description** |         The description to be shown<br /> `string`         |       - |
| **options**     | Any additional props to be passed to the item<br /> `Type` |       - |

Here is an example when specifying the `options` prop

```typescript
interface MyCustomProps {
    isExternal: boolean;
}

const linkListItems: LinkListItemProps<MyCustomProps> = [
    {
        title: "First item",
        description: "This is the first",
        options: { isExternal: false },
    },
    {
        title: "Second item",
        description: "This is the second",
        options: { isExternal: true },
    },
];
```
