# Usage

There are a few ways to make use of the Overlay component.

## Local use

This applies to cases when you intend to use the `Overlay` in a selected
component(s).

> Note: This might not work properly if you have elements of a higher z-index
> that is outside of the component that you are specifying the `Overlay` with.
> as the styling is relative to its parent container.

Simply specify it alongside the component.

```tsx
const SomeComponent = () => {
    const [show, setShow] = useState(false);

    const openModal = () => setShow(true);
    const closeModal = () => setShow(true);

    return (
        <Container>
            <Button.Default onClick={openModal}>Open</Button.Default>
            <Overlay visible={show} onOverlayClick={closeModal} {...args}>
                <SimpleModal show={show} />
            </Overlay>
        </Container>
    );
};
```

## Global use

This applies when you intend to use the `Overlay` in the root level (i.e. `<body>`)

To use, specify a placeholder element in `body`

```html
<html>
    <body>
        <!-- your content here -->
        <div id="overlay-root" />
    </body>
</html>
```

Then pass the id into the `root` prop of the `Overlay`.

```tsx
const SomeComponent = () => {
    const [show, setShow] = useState(false);

    const openModal = () => setShow(true);
    const closeModal = () => setShow(true);

    return (
        <Container>
            <Button.Default onClick={openModal}>Open</Button.Default>
            <Overlay
                visible={show}
                onOverlayClick={closeModal}
                root="overlay-root"
                {...args}
            >
                <SimpleModal show={show} />
            </Overlay>
        </Container>
    );
};
```
