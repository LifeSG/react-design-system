## Props

| Name     | Description                                                              | Default |
| -------- | ------------------------------------------------------------------------ | ------- |
| items`*` | The custom timeline item to be displayed. <br/> `TimelineItemProps`      | -       |
| startCol | The column at which the time line items should start at.<br /> number    | 3       |
| colSpan  | The number of columns that the time line items should span.<br /> number | 8       |

## Timeline items

Each item on the timeline is passed in with the properties defined in `TimelineItemProps`

| Name       | Description                                                                                                                                                                         | Default |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------- |
| title`*`   | The text title of the timeline item. <br /> `string`                                                                                                                                | -       |
| content`*` | The content of the timeline item to be rendered after the title. Contains any call to action. Formatting within the content is maintained by the user <br /> `string` `JSX.Element` | -       |
| statuses   | The status pills to be rendered under the title <br /> `TimelineStatusProps`                                                                                                        | -       |

```tsx
{
	title: "Timeline title",
	content: (
		<>
			<Text.Body>
				Lorem ipsum dolor sit amet, consectetur.
			</Text.Body>
			<Button.Default>Action Button</Button.Default>
		</>
	),
}
```

## Timeline status

These are the status pills rendered under the title. There can only be a maximum of `two` status pills.
Properties of the status pills are defined in `TimelineStatusProps`

| Name     | Description                                           | Default |
| -------- | ----------------------------------------------------- | ------- |
| type`*`  | The colour of the status pill.<br /> `dark` `light`   | `light` |
| label`*` | The text is render on the status pill <br /> `string` | -       |

```typescript
statuses: [
    { type: "light", label: "Status light" },
    { type: "dark", label: "Status dark" },
];
```
