# Elements Usage

The VideoBanner comes with some supporting elements that you can use
when specifying `type=custom` for the `VideoBanner`.

## Title

This is the heading text that tells of the title of the `VideoBanner`.

### Props

This component inherits the [Heading](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/Heading_Elements) attributes.

> Title is using the `h3` element.

### Usage

You will specify this component in the `content.children` prop of the `VideoBanner` component.

```
<VideoBanner.Wrapper
    type="custom"
    videoSrc="https://youtu.be/2Gg6Seob5Mg"
	mobileBackgroundImageSrc={require("../assets/sample-video-banner-image.jpg")}
    content={{
        children: <VideoBanner.Title>My sample title</VideoBanner.Title>,
    }}
/>
```

### Styling

You can also customise the style of this component by specifying the `className` or if you are using `Styled Components`
you can extend from this component.

> This component allows a maximum of **3 lines**. Thereafter it would be truncated with ellipsis (`...`).

## Description

This component works as a `<span>` element, hence it inherits all the `HTMLElement` props.

## Action

This is the action button that allows a custom action to happen when clicked.

### Props

This component inherits the [HTMLAnchorElement](https://developer.mozilla.org/en-US/docs/Web/API/HTMLAnchorElement) attributes.

In addition there is a `type` prop that you can specify with 2 possible values.

-   `default` - comes with a default margin top style
-   `custom` - no style

### Usage

You will specify this component in the `content.children` prop of the `VideoBanner` component.

```
<VideoBanner.Wrapper
    type="custom"
    videoSrc="https://youtu.be/2Gg6Seob5Mg"
    mobileBackgroundImageSrc={require("../assets/sample-video-banner-image.jpg")}
    content={{
        children: (
            <div style={{ display: "flex", flexDirection: "column" }}>
                <VideoBanner.Title>My sample title</VideoBanner.Title>
                <VideoBanner.ActionButton onClick={someFn}>
					Click on me
				</VideoBanner.ActionButton>
            </div>
        ),
    }}
/>
```

<VideoBanner.Wrapper
    type="custom"
    videoSrc="https://youtu.be/2Gg6Seob5Mg"
    mobileBackgroundImageSrc={require("./assets/sample-video-banner-image.jpg")}
    content={{
        children: (
            <div style={{ display: "flex", flexDirection: "column" }}>
                <VideoBanner.Title>My sample title</VideoBanner.Title>
                <VideoBanner.ActionButton>Click on me</VideoBanner.ActionButton>
            </div>
        ),
    }}
/>

### Styling

You can also customise the style of this component by specifying the `className` or if you are using `Styled Components`
you can extend from this component.

> This component comes with a default `margin-top: 2.5rem`. To not have this style, you can either override the
> style as prescribed above, or specify the `type="custom"`.
