# Modifying Text styles

In some cases, you might need to modify the styles of the Text component. You can simply do so as such:

## Using `styled-components`

```tsx
import { Text } from "react-lifesg-design-system";

const ModifiedText = styled(Text.Body)`
    color: red;
    margin-bottom: 2rem;
`;
```

## Using with CSS

```tsx
import { Text } from "react-lifesg-design-system";

<Text.Body className="my-custom-class">Hello World</Text.Body>;
```
