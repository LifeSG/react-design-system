<Title>Accordion</Title>

<Secondary>Overview</Secondary>

Displays expandable and collapsible content panels that can be toggled by the user.

```tsx
import { Accordion } from "@lifesg/react-design-system/accordion";
```

<Canvas of={AccordionStories.Default} />

<Heading3>Smaller Accordion item titles</Heading3>

Specifying the type in the `Accordion.Item` produces a smaller text size for each item's title.

<Canvas of={AccordionStories.SmallAccordionItemTitle} />

<Heading3>Showing title in mobile view</Heading3>

By default, the `Accordion` will not render the title in mobile views. However if you require, you may
use the `showTitleInMobile` prop to enable the title to be visible in mobile view.

<Canvas of={AccordionStories.TitleShownInMobile} />

<Heading3>No title</Heading3>

If you do not require a title in the `Accordion`, simply omit the `title` prop.

<Canvas of={AccordionStories.NoTitle} />

<Heading3>No Expand/Collapse</Heading3>

If you do not require a expand/collapse button in the `Accordion`, use `enableExpandAll={false}`.

<Canvas of={AccordionStories.NoExpandCollapse} />
