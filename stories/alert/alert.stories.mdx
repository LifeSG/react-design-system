<Secondary>Overview</Secondary>

The primary call to action component.

```tsx
import { Alert } from "@lifesg/react-design-system/alert";
```

<Canvas>
    <Story name="Alert">
        <>
            <Alert type="error">
                This will notify users of an error that has occurred.
            </Alert>
            <br />
            <Alert type="warning">
                This will serves as a warning or disclaimer to users.
            </Alert>
            <br />
            <Alert type="success">
                This will serves as a acknowledgement of a successful action.
            </Alert>
            <br />
            <Alert
                type="warning"

<Heading3>Advanced styling</Heading3>

If you require text of a bolder weight, you can make use of the `<strong>` tag.

<Canvas>
    <Story name="With bold text">

                    hyperlink
                </a>
                &nbsp;to direct users to some external source.

</Canvas>
