# Animations

## Overview

A component that renders an animation on the UI.

```tsx
import { <Component_name> } from "@lifesg/react-design-system/animations";
```

## Usage

```tsx
import { LoadingSpinner } from "@lifesg/react-design-system/animations";

export const SomeComponent = () => {
    return (
        <div>
            <LoadingSpinner />
        </div>
    );
};
```
