# Animations

## Overview

A component that renders an animation on the UI.

```tsx
import { <Component_name> } from "@lifesg/react-design-system/animations";
```

## Themed loading spinner

The `ThemedLoadingSpinner` automatically adapts to your theme's
`resourceScheme`.

<Canvas of={AnimationStories.ThemedAnimation} />

## Collection

If you need to render a specific animation regardless of theme, you can use the
standalone component.

## Customisable animations

<DocNote>
    Customisation is only supported by the animations in this section.
</DocNote>

### LoadingDotsSpinner

```tsx
import { LoadingDotsSpinner } from "@lifesg/react-design-system/animations";
```

If no color is specified, this component uses the
`Animation["loading-dots-spinner-colour"]` token.

<Canvas of={AnimationStories.CustomLoadingDotsSpinner} />

### Component tokens

This component can be customised via the `Animation` tokens.

<Story of={AnimationStories.TokenCustomisation} />

## Usage

```tsx
import { LoadingSpinner } from "@lifesg/react-design-system/animations";

export const SomeComponent = () => {
    return (
        <div>
            <LoadingSpinner />
        </div>
    );
};
```
