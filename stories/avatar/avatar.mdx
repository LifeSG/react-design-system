# Avatar

## Overview

Avatar component that displays a user avatar with an optional badge and dropdown menu.
Commonly used with the Navbar to represent a user's profile, but can also be used standalone.

```tsx
import { Avatar } from "@lifesg/react-design-system/avatar";
```
