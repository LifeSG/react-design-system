# BoxContainer

## Overview

Displays contents in a box styling layout that can be collapsible if specified.

```tsx
import { BoxContainer } from "@lifesg/react-design-system/box-container";
```

## With a display state

Specifying the display state allows you to render an icon alongside the title. It is good to indicate if the contents
of the `BoxContainer` has some special attention required.

There are currently 3 states:

-   `default` - where no icon will be rendered
-   `error` - a red icon
-   `warning` - an orange icon

In this example, we are using the `error` display state.
