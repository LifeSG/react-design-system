# Calendar

## Overview

Displays a calendar component.

```tsx
import { Calendar } from "@lifesg/react-design-system/calendar";
```

<Canvas of={CalendarStories.Default} />

## With disabled dates

You can use `disabledDates` to specify dates that are not available for selection.

_(In this example,2 days before today and 2 days after today are disabled)_

<Canvas of={CalendarStories.WithDisabledDates} />

## Within minimum and maximum dates

You can specify the `minDate` and/or `maxDate` to control the selection range.

<Canvas of={CalendarStories.MinAndMaxDates} />

## Allow selection of disabled dates

When `allowDisabledSelection` is set, dates are visually disabled, but still selectable.

<Canvas of={CalendarStories.AllowDisabledSelection} />

## Show active month days only

You can use `showActiveMonthDaysOnly` to display only dates for the selected month.

<Canvas of={CalendarStories.ShowActiveMonthDaysOnly} />
