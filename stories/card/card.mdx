# Card

## Overview

A content container that has rounded corners and appears elevated.

```tsx
import { Card } from "@lifesg/react-design-system/card";
```

<Canvas of={CardStories.Default} />

## Custom content

You can construct a custom layout and pass it into the component.

<Canvas of={CardStories.CustomContent} />

## Customising the style

You can customize/override the style of the `Card` component by extending it:

```tsx
import { Card } from "@lifesg/react-design-system/card";
import styled from "styled-components";

export const StyledCard = styled(Card)`
    max-width: 30rem;
    /* Other CSS properties... */
`;
```
