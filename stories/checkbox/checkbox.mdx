<Title>Checkbox</Title>

<Secondary>Overview</Secondary>

A field that allows a user to enter a response by clicking, and shown with a tick (checked) when
activated.

```tsx
import { Checkbox } from "@lifesg/react-design-system/checkbox";
```

There are 2 display sizes available for the `Checkbox`.

-   Default
-   Small

<Canvas of={CheckboxStories.Default} />

<Heading3>Indeterminate state</Heading3>

A common use case for the indeterminate state is to represent the partial selection of sub-options.
