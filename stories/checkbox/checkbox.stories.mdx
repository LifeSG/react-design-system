<Title>Checkbox</Title>

<Secondary>Overview</Secondary>

A field that allows a user to enter a response by clicking, and shown with a tick (checked) when
activated.

```tsx
import { Checkbox } from "@lifesg/react-design-system/checkbox";
```

There are 2 display sizes available for the `Checkbox`.

-   Default
-   Small

<Canvas>
    <Story name="Checkbox">
        {() => {
            const [checked1, setChecked1] = useState(false);
            const [checked2, setChecked2] = useState(false);
            return (
                <Container>
                    <Checkbox
                        displaySize="default"
                        checked={checked1}
                        onClick={() => setChecked1(!checked1)}
                    />
                    <Checkbox
                        displaySize="small"
                        checked={checked2}
                        onClick={() => setChecked2(!checked2)}
                    />
                </Container>
            );
        }}
    </Story>
</Canvas>
