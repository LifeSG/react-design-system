<Secondary>Overview</Secondary>

A field that allows a user to enter a response by clicking, and shown with a tick (checked) when
activated.

```tsx
import { Checkbox } from "@lifesg/react-design-system/checkbox";
```

There are 2 display sizes available for the `Checkbox`.

-   Default
-   Small

<Heading3>Indeterminate state</Heading3>

A common use case for the indeterminate state is to represent the partial selection of sub-options.

<Canvas>
    <Story name="Indeterminate state">
        {() => {
            const [checked1, setChecked1] = useState(true);
            const [checked2, setChecked2] = useState(false);
            return (
                <>
                    <Checkbox
                        checked={checked1 && checked2}
                        indeterminate={checked1 !== checked2}
                        onChange={() => {
                            if (checked1 !== checked2) {
                                setChecked1(true);
                                setChecked2(true);
                            } else {
                                setChecked1(!checked1);
                                setChecked2(!checked1);
                            }
                        }}
                    />
                    <SubOption>
                        <Checkbox
                            checked={checked1}
                            onChange={() => setChecked1(!checked1)}
                        />
                        Sub-option 1
                    </SubOption>
                    <SubOption>
                        <Checkbox
                            checked={checked2}
                            onChange={() => setChecked2(!checked2)}
                        />
                        Sub-option 2
                    </SubOption>
                </>
            );
        }}
    </Story>
</Canvas>
