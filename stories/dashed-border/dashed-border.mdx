# DashedBorder

## Overview

`DashedBorder` is a lightweight wrapper component for rendering a dashed border
container using theme tokens.

```tsx
import { DashedBorder } from "@lifesg/react-design-system/dashed-border";
```

## Basic Usage

Use it as a container around any content.
