## Overview

A component that displays the current date and day, with support for navigating
the dates.

```tsx
import { DateNavigator } from "@lifesg/react-design-system/date-navigator";
```

<Canvas of={DateNavigatorStories.Default} />

## With navigation

Navigation buttons can be enabled by specifying the `onLeftArrowClick` and
`onRightArrowClick` callbacks.

<Canvas of={DateNavigatorStories.WithNavigation} />

## Loading state

You can specify the `loading` prop to disable date navigation while asynchronous
tasks are running.

<Canvas of={DateNavigatorStories.LoadingState} />

## With date range

The corresponding navigation button is disabled when the current date is \<=
`minDate` or \>= `maxDate`.
