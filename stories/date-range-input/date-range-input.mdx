<Title>Date Range Input</Title>

<Secondary>Overview</Secondary>

Linked start and end date inputs that open a shared calendar overlay for selecting a date range. Supports week, month, and fixed-day range variants in addition to free-range mode.

```tsx
import { DateRangeInput } from "@lifesg/react-design-system/date-range-input";
```
