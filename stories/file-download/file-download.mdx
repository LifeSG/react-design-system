# FileDownload

## Overview

A component that allows for downloading files.

```tsx
import { FileDownload } from "@lifesg/react-design-system/file-download";
```

<Canvas of={FileDownloadStories.Default} />

## With error display

An error will be displayed when the file download has failed.

<Canvas of={FileDownloadStories.ErrorDisplay} />

A custom error message can be specified for each item.

<Canvas of={FileDownloadStories.WithCustomError} />

## Download readiness

The `ready` prop allows users to manage the availability of a file.

This can be used in scenarios where an image needs time to be generated and is not immediately available.

<Canvas of={FileDownloadStories.DownloadReadiness} />

## Default text styling

The title and description support styling for commonly used markup such as `<strong>` or `<a>`.
