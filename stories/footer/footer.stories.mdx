<Secondary>About disclaimer links</Secondary>

As the disclaimer links are mandated to be in the `Footer`, the component already comes with a predefined set of values.

<Heading3>Predefined values</Heading3>

|            | privacy                                   | termsOfUse                           | reportVulnerability                      |
| ---------- | ----------------------------------------- | ------------------------------------ | ---------------------------------------- |
| `children` | Privacy statement                         | Terms of use                         | Report vulnerability                     |
| `href`     | https://www.life.gov.sg/privacy-statement | https://www.life.gov.sg/terms-of-use | https://tech.gov.sg/report_vulnerability |
| `rel`      | `noopener`                                | `noopener`                           | `noopener`                               |
| `external` | `false`                                   | `false`                              | `true`                                   |

> Refer to the Component API table below for more information on the properties

<br />
<Heading3>Customising the links</Heading3>

You are given the flexibility to customise any of the links **except the link text**. You can do so as such:

```tsx
import { DisclaimerLinks, Footer } from "@lifesg/react-design-system/footer";

const disclaimerLinks: DisclaimerLinks = {
    reportVulnerability: {
        href: "https://tech.gov.sg/report_vulnerability",
        target: "_blank",
        rel: "noopener",
        external: true,
    },
};

return (
	<Footer disclaimerLinks={disclaimerLinks} />;
);
```
