<Title>Form.DateInput</Title>

<Secondary>Overview</Secondary>

The field that allows a user to enter a date value.

```tsx
import { Form } from "@lifesg/react-design-system/form";
```

<Heading3>Using a default date input</Heading3>

<Canvas>
    <Story name="DateInput">
        <StoryContainer>
            <Container>
                <Form.DateInput label="This is the default date input" />
                <Form.DateInput
                    label="This is the date input without button"
                    withButton={false}
                />
                <Form.DateInput label="This is the disabled state" disabled />
                <Form.DateInput label="This is the readOnly state" readOnly />
            </Container>
        </StoryContainer>
    </Story>
</Canvas>

<Heading3>Using a range selection</Heading3>

You can use `variant='range'` to specify to range selection.

<Canvas>
    <Story name="Range input">
        {() => {
            const [startValue] = useState(dayjs().day(10).format("YYYY-MM-DD"));
            const [endValue] = useState(dayjs().day(20).format("YYYY-MM-DD"));
            return (
                <StoryContainer>
                    <Container>
                        <Form.DateInput
                            label="This is the default range date input"
                            variant="range"
                        />
                        <Form.DateInput
                            label="This is the range date input without button"
                            variant="range"
                            withButton={false}
                            value={startValue}
                            endValue={endValue}
                        />
                        <Form.DateInput
                            label="This is the disabled state"
                            disabled
                            variant="range"
                            value={startValue}
                            endValue={endValue}
                        />
                        <Form.DateInput
                            label="This is the readOnly state"
                            readOnly
                            variant="range"
                            value={startValue}
                            endValue={endValue}
                        />
                    </Container>
                </StoryContainer>
            );
        }}
    </Story>
</Canvas>

<Heading3>With disabled dates</Heading3>

You can use `disabledDates` to specify dates that are not available for selection.

<Canvas>
    <Story name="With disabled dates">
        {() => {
            const [disabledDates] = useState([
                dayjs().day(10).format("YYYY-MM-DD"),
                dayjs().day(15).format("YYYY-MM-DD"),
                dayjs().day(25).format("YYYY-MM-DD"),
            ]);
            return (
                <StoryContainer>
                    <Container>
                        <Form.DateInput
                            label="This date input with disabled dates"
                            disabledDates={disabledDates}
                        />
                        <Form.DateInput
                            label="This date input with disabled dates"
                            variant="range"
                            disabledDates={disabledDates}
                        />
                    </Container>
                </StoryContainer>
            );
        }}
    </Story>
</Canvas>

<Heading3>Within date range</Heading3>

You can restrict selection to a date range using `between`.

<Canvas>
    <Story name="Within date range">
        {() => {
            const [between] = useState([
                dayjs().day(7).format("YYYY-MM-DD"),
                dayjs().day(21).format("YYYY-MM-DD"),
            ]);
            return (
                <StoryContainer>
                    <Container>
                        <Form.DateInput
                            label="This date input with restrict selection dates"
                            between={between}
                        />
                        <Form.DateInput
                            label="This date input with restrict selection dates"
                            variant="range"
                            between={between}
                        />
                    </Container>
                </StoryContainer>
            );
        }}
    </Story>
</Canvas>

<Heading3>Using the field as a standalone</Heading3>

In the case that you require the date input field as a standalone, you can do this.

```tsx
import { DateInput } from "@lifesg/react-design-system/date-input";
```
