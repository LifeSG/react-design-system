<Title>Form.DateInput</Title>

<Secondary>Overview</Secondary>

A field that allows a user to enter a single date.

```tsx
import { Form } from "@lifesg/react-design-system/form";
```

<Heading3>Using a default date input</Heading3>

<Canvas>
    <Story name="DateInput">
        <StoryContainer>
            <Form.DateInput label="This is the default date input" />
            <Form.DateInput
                label="This is the date input without buttons"
                withButton={false}
            />
            <Form.DateInput label="This is the disabled state" disabled />
            <Form.DateInput label="This is the readonly state" readOnly />
            <Form.DateInput
                label="This is the error state"
                errorMessage="Invalid date"
            />
        </StoryContainer>
    </Story>
</Canvas>

<Heading3>With disabled dates</Heading3>

You can use `disabledDates` to specify dates that are not available for selection.

<Canvas>
    <Story name="With disabled dates">
        {() => {
            const [disabledDates] = useState([
                dayjs().date(10).format("YYYY-MM-DD"),
                dayjs().date(15).format("YYYY-MM-DD"),
                dayjs().date(25).format("YYYY-MM-DD"),
            ]);
            return (
                <StoryContainer>
                    <Form.DateInput
                        label="This is a date input with disabled dates"
                        disabledDates={disabledDates}
                    />
                </StoryContainer>
            );
        }}
    </Story>
</Canvas>

<Heading3>Within minimum and maximum dates</Heading3>

You can specify the `minDate` and/or `maxDate` to control the selection range.

<Canvas>
    <Story name="Min and max dates">
        {() => {
            return (
                <StoryContainer>
                    <Container>
                        <Form.DateInput
                            label="This date input has restricted selection"
                            minDate={dayjs()
                                .subtract(2, "days")
                                .format("YYYY-MM-DD")}
                            maxDate={dayjs()
                                .add(20, "days")
                                .format("YYYY-MM-DD")}
                        />
                    </Container>
                </StoryContainer>
            );
        }}
    </Story>
</Canvas>

<Heading3>Allow selection of disabled dates</Heading3>

When `allowDisabledSelection` is set, dates are visually disabled, but still selectable.

<Canvas>
    <Story name="Allow disabled selection">
        {() => {
            const [disabledDates] = useState([
                dayjs().date(10).format("YYYY-MM-DD"),
                dayjs().date(15).format("YYYY-MM-DD"),
                dayjs().date(25).format("YYYY-MM-DD"),
            ]);
            return (
                <StoryContainer>
                    <Container>
                        <Form.DateInput
                            label="This date input allows selection of disabled dates"
                            disabledDates={disabledDates}
                            minDate={dayjs()
                                .subtract(6, "weeks")
                                .format("YYYY-MM-DD")}
                            maxDate={dayjs()
                                .add(6, "weeks")
                                .format("YYYY-MM-DD")}
                            allowDisabledSelection
                        />
                    </Container>
                </StoryContainer>
            );
        }}
    </Story>
</Canvas>

<Heading3>Using the field as a standalone</Heading3>

In the case that you require the date input field as a standalone, you can do this.

```tsx
import { DateInput } from "@lifesg/react-design-system/date-input";
```
