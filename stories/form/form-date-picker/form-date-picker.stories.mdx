<Secondary>Overview</Secondary>

The field that allows a user to enter a date value.

```tsx
import { Form } from "@lifesg/react-design-system/form";
```

> **Important note**: This component currently does not do date value validation

<Heading3>Using the field as a standalone</Heading3>

In the case that you require the date calendar field as a standalone, you can do this.

```tsx
import { DatePicker } from "@lifesg/react-design-system/date-picker";
```
