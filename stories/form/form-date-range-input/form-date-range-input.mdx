<Title>DateRangeInput</Title>

<Secondary>Overview</Secondary>

A field that allows a user to enter a date range.

```tsx
import { Form } from "@lifesg/react-design-system/form";
```

<Canvas of={FormDateRangeInputStories.Default} />

<Heading3>Week Input</Heading3>

A field that only allows user to pick a week date.

<Canvas of={FormDateRangeInputStories.WeekInput} />

<Heading3>Fixed range input</Heading3>

A field that only allows a user to enter a date range with a fixed duration.

<Canvas of={FormDateRangeInputStories.FixedRangeInput} />

<Heading3>With disabled dates</Heading3>

You can use `disabledDates` to specify dates that are not available for selection.

_(In this example, 2 days before today and 2 days after today are disabled)_

<Canvas of={FormDateRangeInputStories.WithDisabledDates} />

<Heading3>Within minimum and maximum dates</Heading3>

You can specify the `minDate` and/or `maxDate` to control the selection range.

<Canvas of={FormDateRangeInputStories.MinAndMaxDates} />

<Heading3>Allow selection of disabled dates</Heading3>

When `allowDisabledSelection` is set, dates are visually disabled, but still selectable.

<Canvas of={FormDateRangeInputStories.AllowDisabledSelection} />

<Heading3>Rendering in grid layouts</Heading3>

You can also specify [ColDiv's](/docs/getting-started-layout-column-divs--column-divs)
column props to span across the specified columns in grid layouts.

<Canvas of={FormDateRangeInputStories.RenderingInGridLayout} />

<Heading3>Using the field as a standalone</Heading3>

In the case that you require the date range input field as a standalone, you can do this.

```tsx
import { DateRangeInput } from "@lifesg/react-design-system/date-range-input";
```

<Canvas of={FormDateRangeInputStories.StandaloneUsage} />
