<Title>DateRangeInput</Title>

<Secondary>Overview</Secondary>

A field that allows a user to enter a date range.

```tsx
import { Form } from "@lifesg/react-design-system/form";
```

<Canvas>
    <Story name="DateRangeInput">
        {() => {
            const [start, setStart] = useState(
                dayjs().subtract(1, "month").date(21).format("YYYY-MM-DD")
            );
            const [end, setEnd] = useState(
                dayjs().date(7).format("YYYY-MM-DD")
            );
            return (
                <StoryContainer>
                    <Form.DateRangeInput
                        label="This is the date range input"
                        value={start}
                        valueEnd={end}
                        onChange={(s, e) => {
                            setStart(s);
                            setEnd(e);
                        }}
                    />
                    <Form.DateRangeInput
                        label="This has no action buttons on desktop viewports"
                        withButton={false}
                    />
                    <Form.DateRangeInput
                        label="This is the disabled state"
                        withButton={false}
                        value="2023-04-20"
                        disabled
                    />
                    <Form.DateRangeInput
                        label="This is the readonly state"
                        withButton={false}
                        value="2023-04-20"
                        readOnly
                    />
                    <Form.DateRangeInput
                        label="This is the error state"
                        withButton={false}
                        errorMessage="Invalid date"
                    />
                </StoryContainer>
            );
        }}
    </Story>
</Canvas>

<Heading3>With disabled dates</Heading3>

You can use `disabledDates` to specify dates that are not available for selection.

<Canvas>
    <Story name="With disabled dates">
        {() => {
            const [disabledDates] = useState([
                dayjs().date(10).format("YYYY-MM-DD"),
                dayjs().date(15).format("YYYY-MM-DD"),
                dayjs().date(25).format("YYYY-MM-DD"),
            ]);
            return (
                <StoryContainer>
                    <Form.DateRangeInput
                        label="This has disabled dates"
                        disabledDates={disabledDates}
                    />
                </StoryContainer>
            );
        }}
    </Story>
</Canvas>

<Heading3>Within date range</Heading3>

You can restrict selection to a date range using `between`.

<Canvas>
    <Story name="Within date range">
        {() => {
            const [between] = useState([
                dayjs().date(7).format("YYYY-MM-DD"),
                dayjs().date(21).format("YYYY-MM-DD"),
            ]);
            return (
                <StoryContainer>
                    <Form.DateRangeInput
                        label="This has restricted selection"
                        between={between}
                    />
                </StoryContainer>
            );
        }}
    </Story>
</Canvas>

<Heading3>Using the field as a standalone</Heading3>

In the case that you require the date range input field as a standalone, you can do this.

```tsx
import { DateRangeInput } from "@lifesg/react-design-system/date-range-input";
```
