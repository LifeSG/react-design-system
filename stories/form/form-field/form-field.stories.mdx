A field that allows a user to enter a response.

```tsx
import { Form } from "@lifesg/react-design-system/form";
```

<Heading3>Using the field independently</Heading3>

In the case that you require the input field as a standalone, you can do this.

```tsx
import { Input } from "@lifesg/react-design-system/input";
```
