<Title>Form.InputMask</Title>

<Secondary>Overview</Secondary>

An enhanced field that allows a user to enter a response that is masked on blur.

```tsx
import { Form } from "@lifesg/react-design-system/form";
```

<Heading3>Masking</Heading3>

You are able to specify the range of index to mask or unmask, the regex to mask or customize the masking logic by passing in a maskTransformer.

<Canvas>
    <Story name="Masking">
        <StoryContainer>
            <Container>
                <Form.InputMask
                    label="Set mask range to [2,5]"
                    value="S1234567D"
                    maskRange={[2, 5]}
                />
                <Form.InputMask
                    label="Set mask range to [2,100]"
                    value="S1234567D"
                    maskRange={[2, 100]}
                />
                <Form.InputMask
                    label="Set mask range to [100,2] (auto-determines start and end index)"
                    value="S1234567D"
                    maskRange={[100, 2]}
                />
                <Form.InputMask
                    label="Set unmask range to [1,4]"
                    value="S1234567D"
                    unmaskRange={[1, 4]}
                />
                <Form.InputMask
                    label="Set mask regex to /\D/g"
                    value="S1234567D"
                    maskRegex={/\D/g}
                />
                <Form.InputMask
                    label="Set mask transformer that replaces non-digit character with *"
                    value="S1234567D"
                    maskTransformer={(value) => value.replace(/\D/g, "*")}
                />
            </Container>
        </StoryContainer>
    </Story>
</Canvas>

<Heading3>Customizations</Heading3>

You are able to make customizations to the mask icon active and inactive color, as well as the mask character.

<Canvas>
    <Story name="With customizations">

                />
                <Form.InputMask

                    value="S1234567D"

                <Form.InputMask
                    label="Overwrite mask and unmask icon"
                    value="S1234567D"
                    maskRange={[2, 5]}
