```tsx
import { Form } from "@lifesg/react-design-system/form";
```

<Canvas>
    <Story name="Label">

                <div style={{ display: "flex", alignItems: "center" }}>

                        <Checkbox
                            checked={checked}

        }}

<Heading3>With addon</Heading3>

Form labels come with an optional addon that you can use by specifying the `addon` prop.

<Canvas>
    <Story name="With addon">
        {() => {
            const [checked, setChecked] = useState(false);
            return (
                <div style={{ display: "flex", alignItems: "center" }}>
                    <div style={{ marginRight: "1.5rem" }}>
                        <Checkbox
                            checked={checked}
                            onClick={() => setChecked(!checked)}
                        />
                    </div>
                    <Form.Label
                        style={{ marginBottom: "0" }}
                        addon={{
                            content:
                                "This is the form label's popover. And this will only be visible if you specify the addon prop",
                            type: "popover",
                        }}
                    >
                        The form label with an addon
                    </Form.Label>
                </div>
            );
        }}
    </Story>
</Canvas>
