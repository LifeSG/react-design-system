<Secondary>Overview</Secondary>

A field that provides a nested set of options for a user to select.

```tsx
import { Form } from "@lifesg/react-design-system/form";
```

<Canvas>
    <Story name="NestedSelect">
        {() => {
            const [selectedKeyPath, setSelectedKeyPath] = useState([
                "32",
                "52",
                "21",
            ]);
            return (
                <StoryContainer>
                    <Container>
                        <Form.NestedSelect
                            label="This is the nested selected field"
                            options={options}
                            selectedKeyPath={selectedKeyPath}
                            onSelectOption={(keyPaths, value) =>
                                setSelectedKeyPath(keyPaths)
                            }
                            placeholder="Select"
                        />
                    </Container>
                </StoryContainer>
            );
        }}
    </Story>
</Canvas>

<Heading3>Using the field as a standalone</Heading3>

In the case that you require the nested select field as a standalone, you can do this.

```tsx
import { InputNestedSelect } from "@lifesg/react-design-system/input-nested-select";
```
