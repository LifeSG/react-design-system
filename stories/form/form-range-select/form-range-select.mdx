<Title>Form.RangeSelect</Title>

<Secondary>Overview</Secondary>

A field that provides a way to select options based on a range.

```tsx
import { Form } from "@lifesg/react-design-system/form";
```

<Canvas of={FormRangeSelectStories.Default} />

<Heading3>With custom list item display</Heading3>

<Canvas of={FormRangeSelectStories.WithCustomListDisplay} />

<Canvas of={FormRangeSelectStories.WithSearch} />

<Heading3>Rendering in grid layouts</Heading3>

column props to span across the specified columns in grid layouts.

<Canvas of={FormRangeSelectStories.RenderingInGridLayout} />

