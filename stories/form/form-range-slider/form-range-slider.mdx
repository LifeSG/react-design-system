# Form.RangeSlider

## Overview

A field that allows a user to select a range of values within a numeric range.

```tsx
import { Form } from "@lifesg/react-design-system/form";
```

<Canvas of={FormRangeSliderStories.RangeSlider} />

## Minimum range

A minimum difference between each value can be specified.

<Canvas of={FormRangeSliderStories.MinimumRange} />

## Color customisation

For advanced styling, each segment of the slider track can be individually colored.

<Canvas of={FormRangeSliderStories.ColorCustomisation} />

## With labels

You can display labels for the min and max values with `showSliderLabels`. A prefix or
suffix can be specified to act as a unit. A render function is also supported for advanced customisation.

You can indicate the current selection with `showIndicatorLabel`.

<Canvas of={FormRangeSliderStories.WithLabels} />

## Rendering in grid layouts

You can also specify [ColDiv's](/docs/getting-started-layout-column-divs--column-divs)
column props to span across the specified columns in grid layouts.

<Canvas of={FormRangeSliderStories.RenderingInGridLayout} />

## Using the field as a standalone

In the case that you require the range slider field as a standalone, you can do this.

```tsx
import { InputRangeSlider } from "@lifesg/react-design-system/input-range-slider";
```
