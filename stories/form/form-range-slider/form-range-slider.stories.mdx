<Title>Form.RangeSlider</Title>

<Secondary>Overview</Secondary>

A field that allows a user to select a range of values within a numeric range.

```tsx
import { Form } from "@lifesg/react-design-system/form";
```

<Canvas>
    <Story name="RangeSlider">
        <StoryContainer>
            <Container>

                    label="This is the range slider"
                    value={[10, 20]}
                />

                    label="This has three thumbs"
                    numOfThumbs={3}

                />

                    label="This is the disabled state"

                    disabled

    </Story>

<Heading3>Minimum range</Heading3>

A minimum difference between each value can be specified.

<Canvas>
    <Story name="Minimum range">
        <StoryContainer>
            <Container>
                <Form.RangeSlider
                    label="This has a min range"
                    value={[0, 30]}
                    minRange={30}
                />
            </Container>
        </StoryContainer>
    </Story>
</Canvas>

<Heading3>Color customisation</Heading3>

Each segment of the slider track can be individually colored.

<Canvas>
    <Story name="Color customisation">
        <StoryContainer>
            <Container>
                <Form.RangeSlider
                    label="This has custom colors"
                    numOfThumbs={4}
                    value={[10, 20, 30, 80]}
                    colors={["red", "orange", "green", "blue", "purple"]}
                />
            </Container>
        </StoryContainer>
    </Story>
</Canvas>

<Heading3>With labels</Heading3>

You can display labels for the min and max values with `showLabels`. A prefix or
suffix can be specified to act as a unit.

A render function is also supported for advanced customisation.

<Canvas>
    <Story name="With labels">
        <StoryContainer>
            <Container>
                <Form.RangeSlider
                    label="With unit as prefix"
                    showLabels
                    labelPrefix="$"
                />
                <Form.RangeSlider
                    label="With unit as suffix"
                    showLabels
                    labelSuffix="km"
                />
                <Form.RangeSlider
                    label="With custom rendering"
                    min={1}
                    max={10}
                    showLabels
                    renderLabel={(val) =>
                        val === 1 ? (
                            <Text.XSmall weight="semibold">1 photo</Text.XSmall>
                        ) : (
                            <Text.XSmall weight="semibold">
                                {val} photos
                            </Text.XSmall>
                        )
                    }
                />
            </Container>
        </StoryContainer>
    </Story>
</Canvas>

<Heading3>Rendering in grid layouts</Heading3>

You can also specify [ColDiv's](/docs/getting-started-layout-column-divs--column-divs)
column props to span across the specified columns in grid layouts.

    <Story
