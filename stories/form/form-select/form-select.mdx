<Title>Form.Select</Title>

<Secondary>Overview</Secondary>

A field that provides a set of options for a user to select.

```tsx
import { Form } from "@lifesg/react-design-system/form";
```

<Canvas of={FormSelectStories.Select} />

<Heading3>With custom list item display</Heading3>

You are able to customise the display of the dropdown list items as well as the selected option.

<Canvas of={FormSelectStories.WithCustomListItemDisplay} />

<Heading3>With search capabilities</Heading3>

<Canvas of={FormSelectStories.WithSearch} />

<Heading3>With custom call to action at the bottom of dropdown list</Heading3>

You are also able to specify a custom call to action component at the bottom
of the list.

<Canvas of={FormSelectStories.WithCustomCallToAction} />

<Heading3>Rendering in grid layouts</Heading3>

You can also specify [ColDiv's](/docs/getting-started-layout-column-divs--column-divs)
column props to span across the specified columns in grid layouts.

<Canvas of={FormSelectStories.RenderingInGridLayout} />

<Heading3>Using the field as a standalone</Heading3>

In the case that you require the select field as a standalone, you can do this.

```tsx

```
