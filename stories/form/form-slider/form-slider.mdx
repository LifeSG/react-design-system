# Form.Slider

## Overview

A field that allows a user to select a value from a numeric range.

```tsx
import { Form } from "@lifesg/react-design-system/form";
```

<Canvas of={FormSliderStories.Slider} />

## Color customisation

For advanced styling, each segment of the slider track can be individually colored.

<Canvas of={FormSliderStories.ColorCustomisation} />

## With labels

You can display labels for the min and max values with `showSliderLabels`. A prefix or
suffix can be specified to act as a unit. A render function is also supported for advanced customisation.

You can indicate the current selection with `showIndicatorLabel`.

<Canvas of={FormSliderStories.WithLabels} />

## Rendering in grid layouts

You can also specify [ColDiv's](/docs/core-layout-coldiv--docs)
column props to span across the specified columns in grid layouts.

<Canvas of={FormSliderStories.RenderingInGridLayout} />

## Using the field as a standalone

In the case that you require the slider field as a standalone, you can do this.

```tsx
import { InputSlider } from "@lifesg/react-design-system/input-slider";
```
