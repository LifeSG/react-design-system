# Form.Textarea

## Overview

A field that allows a user to enter a multi-line response. This is useful if there is a sizeable amount
of free-form text.

```tsx
import { Form } from "@lifesg/react-design-system/form";
```

<Canvas of={FormTextareaStories.Default} />

## Textarea with custom counter label

<Canvas of={FormTextareaStories.WithCustomCounter} />

```tsx
const renderCustomCounter = (maxLength: number, currentValueLength: number) => {
    return (
        <CustomCounterLabel>
            {currentValueLength} / {maxLength}
        </CustomCounterLabel>
    );
};
```
