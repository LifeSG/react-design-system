The field that allows a user to enter a time range.


## Combobox variant

You can also use a `combobox` variant that allows users to either select a time
from a dropdown list of options, or manually type in a time which is
automatically parsed and converted into a valid time format (h:mma for `12hr`,
hh:mm for `24hr`).

This variant comes in built-in validation for unrecognised time format or
invalid time range (start is greater than end).
