<Secondary>Overview</Secondary>

The field that allows a user to enter a time value.

```tsx
import { Form } from "@lifesg/react-design-system/form";
```

<Canvas>
    <Story name="TimeRangePicker">
        {() => {
            const [time1, setTime1] = useState({
                startTime: "",
                endTime: "",
            });
            const timeVal = {
                startTime: "12:00am",
                endTime: "12:30am",
            };
            const [time2, setTime2] = useState({
                startTime: "",
                endTime: "",
            });
            const [time3, setTime3] = useState({
                startTime: "",
                endTime: "",
            });
            return (
                <StoryContainer>
                    <Container>
                        <Form.TimeRangePicker
                            label="This is a TimeRangePicker"
                            value={time1}
                            swapInvalidRange
                            onChange={(value) => setTime1(value)}
                        />
                        <Form.TimeRangePicker
                            label="This is the disabled state"
                            disabled={true}
                        />
                        <Form.TimeRangePicker
                            label="This is the error state"
                            value={time3}
                            onChange={(value) => setTime3(value)}
                            error={true}
                            swapInvalidRange
                            errorMessage="Time input required"
                        />
                        <Form.TimeRangePicker
                            label="This is read only state for timepicker"
                            value={timeVal}
                            readOnly={true}
                        />
                    </Container>
                </StoryContainer>
            );
        }}
    </Story>
</Canvas>

<Heading3>Using the field as a standalone</Heading3>

In the case that you require the timepicker field as a standalone, you can do this.

```tsx
import { TimeRangePicker } from "@lifesg/react-design-system/TimeRangePicker";
```

<Preview>
    <Story name="Standalone usage">
        {() => {
            const [time, setTime] = useState("");
            return (
                <StoryContainer>
                    <Container>
                        <TimeRangePicker
                            value={time}
                            onChange={(value) => setTime(value)}
                        />
                    </Container>
                </StoryContainer>
            );
        }}
    </Story>
</Preview>
