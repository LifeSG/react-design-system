# FullscreenImageCarousel

## Overview

An overlay that display images in a carousel and offers the ability to zoom into the current image.

```tsx
import { FullscreenImageCarousel } from "@lifesg/react-design-system/fullscreen-image-carousel";
```

## With custom content

Slides can render arbitrary content via `renderContent`. If the content is not an image, you are recommended to set `itemLabel` so that the type of content can be described accurately to screen readers.
