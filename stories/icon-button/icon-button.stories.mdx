<Secondary>Overview</Secondary>

A call to action component with an icon in it.

```tsx
import { IconButton } from "@lifesg/react-design-system/icon-button";
```

<Heading3>Highlight</Heading3>

By default, the button is highlighted when focused. This can be customised with the `focusHighlight` prop.

<Preview>
    <Story name="Highlight">
        <IconButton focusHighlight={false}>
            <LockIcon />
        </IconButton>
    </Story>
</Preview>

<Heading3>Outline</Heading3>

By default, the button is not outlined when focused. To defer to the browser's default style, specify the `focusOutline` prop.

(Try focusing the button below with the Tab key)

<Preview>
    <Story name="Outline">
        <IconButton focusOutline="browser">
            <LockIcon />
        </IconButton>
    </Story>
</Preview>

<Heading3>Using custom components</Heading3>

You can also pass in custom components to the `IconButton`. In this example, we will pass in an image.
