<Title>Input Multi Select</Title>

<Secondary>Overview</Secondary>

A multi-option dropdown selector with checkboxes that allows the user to choose multiple items from a list. Supports async loading and search. For single-item selection use `InputSelect`.

```tsx
import { InputMultiSelect } from "@lifesg/react-design-system/input-multi-select";
```
