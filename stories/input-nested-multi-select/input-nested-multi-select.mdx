<Title>Input Nested Multi Select</Title>

<Secondary>Overview</Secondary>

A hierarchical dropdown selector for choosing multiple values from a three-level category tree (L1 → L2 → L3). Supports search and async loading. For single-value selection use `InputNestedSelect`.

```tsx
import { InputNestedMultiSelect } from "@lifesg/react-design-system/input-nested-multi-select";
```
