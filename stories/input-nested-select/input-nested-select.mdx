<Title>Input Nested Select</Title>

<Secondary>Overview</Secondary>

A hierarchical dropdown selector for choosing a single value from a three-level category tree (L1 → L2 → L3). Supports search and async loading. For multiple selection use `InputNestedMultiSelect`.

```tsx
import { InputNestedSelect } from "@lifesg/react-design-system/input-nested-select";
```
