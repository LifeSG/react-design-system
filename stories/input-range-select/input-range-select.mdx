<Title>Input Range Select</Title>

<Secondary>Overview</Secondary>

A pair of linked dropdown selectors for choosing a start and end value from the same option list. Commonly used for filtering by a value range such as price or age.

```tsx
import { InputRangeSelect } from "@lifesg/react-design-system/input-range-select";
```
