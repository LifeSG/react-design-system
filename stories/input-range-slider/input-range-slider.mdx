<Title>Input Range Slider</Title>

<Secondary>Overview</Secondary>

A dual-thumb slider for selecting a numeric interval within a defined range. Supports configurable track colors, indicator labels, and custom label rendering. For a single-value slider use `InputSlider`.

```tsx
import { InputRangeSlider } from "@lifesg/react-design-system/input-range-slider";
```
