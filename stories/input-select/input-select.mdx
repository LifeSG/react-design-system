<Title>Input Select</Title>

<Secondary>Overview</Secondary>

A single-option dropdown selector that allows the user to choose one item from a list. Supports async loading, search, read-only mode, and custom value rendering. For multiple selection use `InputMultiSelect`.

```tsx
import { InputSelect } from "@lifesg/react-design-system/input-select";
```
