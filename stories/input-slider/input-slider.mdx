<Title>Input Slider</Title>

<Secondary>Overview</Secondary>

A single-thumb slider for selecting a numeric value within a defined range. Supports configurable track colors, indicator labels, and custom label rendering. For an interval slider use `InputRangeSlider`.

```tsx
import { InputSlider } from "@lifesg/react-design-system/input-slider";
```
