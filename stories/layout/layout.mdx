# Layout

## Overview

The layout components handle responsive displays for page layouts.

```tsx
import { Layout } from "@lifesg/react-design-system/layout";
```

## Anatomy

The style specifications of the layout container are as such:

| Attribute        | Value                                                                    |
| ---------------- | ------------------------------------------------------------------------ |
| max width        | `1440px`                                                                 |
| horizontal inset | see [breakpoint tokens](/docs/foundations-breakpoint-introduction--docs) |
| vertical inset   | `0`                                                                      |

## Flex column layout

You can arrange content vertically with the `type` prop set to `"flex-column"`.

<Canvas of={LayoutStories.FlexColumnLayout} />

## Grid layout

Setting the `type` prop to `"grid"` configures the container as a [CSS
Grid](https://css-tricks.com/snippets/css/complete-guide-grid/) parent. The
column count and gutters are based on the [breakpoint
tokens](/docs/foundations-breakpoint-introduction--docs).
