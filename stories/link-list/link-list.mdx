# LinkList

## Overview

A component that displays a continuous group of link elements.

```tsx
import { LinkList } from "@lifesg/react-design-system/link-list";
```

<Canvas of={LinkListStories.Default} />

<Canvas of={LinkListStories.Small} />

## Minimizing the list

You are able to specify the number of items to show in the minimized view by using the `maxShown` property. Specifying
this property will allow you to show only the specified number of items while minimizing the rest.
