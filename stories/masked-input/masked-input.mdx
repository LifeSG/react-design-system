<Title>Masked Input</Title>

<Secondary>Overview</Secondary>

A text input that hides sensitive values by default with a toggle button to reveal or conceal the content. Supports custom mask/unmask icons, color customisation, and an async load state for values fetched on demand.

```tsx
import { MaskedInput } from "@lifesg/react-design-system/masked-input";
```
