# Masthead

## Overview

A mandatory banner to be displayed at the top of the page on every .gov.sg digital services. For more info,
visit the [Singapore Government Design System](https://designsystem.tech.gov.sg/components/official-government-banner).

```tsx
import { Masthead } from "@lifesg/react-design-system/masthead";
```

The masthead is a styled version from the original and is integrated with the
[Navbar](/docs/navigation-navbar--docs) as well. The horizontal spacing is
customised to align with the [Layout](/docs/core-layout--docs) container.

<Canvas of={MastheadStories.Default} />

> The component also uses a custom script. Should you encounter a content security
> policy error, do add this script in the `scriptSrc` blob.
