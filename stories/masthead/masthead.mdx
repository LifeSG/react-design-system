<Secondary>Overview</Secondary>

A mandatory banner to be displayed at the top of the page on every .gov.sg digital services. For more info,
visit the [Singapore Government Design System](https://designsystem.tech.gov.sg/components/official-government-banner).

```tsx
import { Masthead } from "@lifesg/react-design-system/masthead";
```

> The masthead is a styled version from the original and is integrated with the [Navbar](/docs/modules-navbar--navbar) as well

<Canvas of={MastheadStories.Default} />
