# Menu

## Overview

A component which appears when hovering or clicking on a target element and
displays a menu. It is commonly used in combination with
[Avatar](/docs/navigation-avatar--docs) and
[Navbar](/docs/navigation-navbar--docs).

This module contains:

-   **Menu** - a utility component that handles the triggering of the menu, extending [PopoverTrigger](/docs/overlays-popoverv2--docs)
-   **Menu.Content** - the main wrapper containing the menu items
-   **Menu.Section** - groups menu items and optionally shows a section label
-   **Menu.Item** - a menu item with optional label and sublabel
-   **Menu.Link** - a menu item that contains a hyperlink

```tsx
import { Menu } from "@lifesg/react-design-system/menu";
```
