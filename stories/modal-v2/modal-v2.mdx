# ModalV2

## Overview

A window or pop up that displays over other page content and provides
information or actions that a user can perform.

```tsx
import { ModalV2 } from "@lifesg/react-design-system/modal-v2";
```

This module contains several components, intended to be used together.

Correct usage of these components ensures the modal will have the correct
appearance and accessible behaviour.

-   `ModalV2` represents the base modal overlay. This is **mandatory**.
-   `ModalV2.Card` represents the default dialog box.
    -   It must be rendered as a child of `ModalV2`
    -   It will only render slots as children

The following slots are available:

-   `ModalV2.CloseButton` is a header containing only the close button
-   `ModalV2.Content` can contain any content
-   `ModalV2.Footer` is a footer containing call-to-actions

## Styling slots

Slots come with default margins, but if custom spacing is required, the styles
can be overridden.

```tsx
const StyledContent = styled(ModalV2.Content)`
    /* replace the margins */
    margin: ${Spacing["spacing-16"]};

    /* or reset them and use your preferred property instead */
    margin: 0;
    padding-inline: ${Spacing["spacing-16"]};
`;
```
