<Title>Notification Toast</Title>

<Secondary>Overview</Secondary>

A component that used to inform users of updates or changes to system status. Communicating with users and providing immediate feedback.

```tsx
import { NotificationToast } from "@lifesg/react-design-system/notification-toast";
```
