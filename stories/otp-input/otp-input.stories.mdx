<Heading3>With a custom action button</Heading3>

You can also customise the action button's props to suit your needs.

<Heading4>
    <strong>Usage of in-built countdown</strong>
</Heading4>

In this instance, you will need to handle the label when a countdown is ongoing.
There are callback functions like `onCooldownStart` and `onCooldownEnd` to assist
you with it.

> You can use `setInterval` to handle the countdown display

<Preview>
    <Story name="With custom action button">
        {() => {
            const numOfInput = 4;
            const [isInCooldown, setIsInCooldown] = useState(true);
            const [otpValues, setOtpValues] = useState(
                new Array(numOfInput).fill("")
            );
            return (
                <StoryContainer>
                    <OtpInput
                        numOfInput={numOfInput}
                        value={otpValues}
                        cooldownDuration={5}
                        onCooldownStart={() => setIsInCooldown(true)}
                        onCooldownEnd={() => setIsInCooldown(false)}
                        onChange={(value) => setOtpValues(value)}
                        actionButtonProps={{
                            styleType: "light",
                            children: isInCooldown
                                ? "Cooling down..."
                                : "My custom button label",
                        }}
                    />
                </StoryContainer>
            );
        }}
    </Story>
</Preview>

<Heading4>
    <strong>Usage of custom countdown mechanism</strong>
</Heading4>

In this instance, you will need to handle the countdown calculation
and rendering entirely on your own. In this example, we will just
use a simple `setTimeout` to handle the change in label.

> You can use `setInterval` to handle the countdown display

<Preview>
    <Story name="With custom action button (self-handle)">
        {() => {
            const [isInCooldown, setIsInCooldown] = useState(true);
            useEffect(() => {
                if (isInCooldown) {
                    setTimeout(() => {
                        setIsInCooldown(false);
                    }, 5000);
                }
            }, [isInCooldown]);
            return (
                <StoryContainer>
                    <OtpInput
                        numOfInput={6}
                        cooldownDuration={0}
                        actionButtonProps={{
                            styleType: "light",
                            children: isInCooldown
                                ? "Cooling down..."
                                : "My custom button label",
                            disabled: isInCooldown,
                            onClick: () => {
                                setIsInCooldown(true);
                            },
                        }}
                    />
                </StoryContainer>
            );
        }}
    </Story>
</Preview>
