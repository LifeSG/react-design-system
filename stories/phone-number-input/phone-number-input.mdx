A combined country code dropdown and phone number text field for structured international phone number entry. Supports a fixed country mode, clear button, and controlled value via a `{ countryCode, number }` object.

```tsx
import { PhoneNumberInput } from "@lifesg/react-design-system/phone-number-input";
```
