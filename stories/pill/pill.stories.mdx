<Title>Pill</Title>

<Secondary>Overview</Secondary>

A status pill component.

Example of usage: As a status indicator on current flow state such as `Pending Approval`, `Approved`, `Rejected` and etc.

```tsx
import { Pill } from "@lifesg/react-design-system/pill";
```

The `Pill` component comes in 2 type, a `solid` and `outline`.

<Heading3>Variations</Heading3>

The `Pill` component comes in 6 colorType, a `black`, `grey`, `green`, `yellow`, `red`, and `blue` color.

Color `black` will be the default colorType.
