<Title>Predictive Text Input</Title>

<Secondary>Overview</Secondary>

An autocomplete input that fetches and displays suggestions as the user types. A minimum character threshold can be configured before fetching begins. Supports keyboard navigation and custom value extractors.

```tsx
import { PredictiveTextInput } from "@lifesg/react-design-system/predictive-text-input";
```
