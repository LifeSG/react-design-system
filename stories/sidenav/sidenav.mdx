# Sidenav

## Overview

A component that renders navigation buttons on the side of a page.

```tsx
import { Sidenav } from "@lifesg/react-design-system/sidenav";
```

## Subitem drawer

The drawer is expanded when a menu item with subitems is hovered or selected. After a subitem is selected, the drawer will close.
