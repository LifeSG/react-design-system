# Table

## Overview

A low-level table component for displaying tabular data. This is a lightweight
alternative to [DataTable](/docs/content-datatable--docs) when advanced styling,
row/column spans, or fully custom rendering is required.

```tsx
import { Table } from "@lifesg/react-design-system/table";
```
