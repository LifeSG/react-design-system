<Title>Tag</Title>

<Secondary>Overview</Secondary>

A static or interactive element that is used to communicate selections, statuses and categories.

```tsx
import { Tag } from "@lifesg/react-design-system/tag";
```

<Heading3>Variations</Heading3>

The `Tag` comes in 2 display formats, `outline` and `solid` and there are 6 color variations that can be denoted using the `colorType` prop.
