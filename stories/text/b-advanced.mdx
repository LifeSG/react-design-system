<Title>Advanced Usage</Title>

<Heading3>Using a combination of styles</Heading3>

In certain use cases, you might need to use multiple `Text` components to achieve the desired result.

<Heading4>Example 1</Heading4>

Including a hyperlink within a set of text.

<PreviewBox>
    {
        <Text.Body>
            The quick brown fox&nbsp;
            <Text.Hyperlink.Default href="https://www.google.com">
                jumps over
            </Text.Hyperlink.Default>
            &nbsp;the lazy dog
        </Text.Body>
    }
</PreviewBox>

<Heading4>Example 2</Heading4>

Including different weights within a set of text. All you need to do is to specify the `inline` prop
for the child components.

<PreviewBox>
    {
        <Text.Body>
            The quick&nbsp;
            <Text.Body weight="light" inline={true}>
                brown fox
            </Text.Body>&nbsp;
            <Text.Body weight="semibold" inline={true}>
                jumps over
            </Text.Body>
            &nbsp;the lazy dog
        </Text.Body>
    }
</PreviewBox>

<Heading3>Paragraph specification</Heading3>

You can include a paragraph styled spacing between text blocks by specifying the `paragraph` prop.

<PreviewBox>
    <Text.Body paragraph={true}>
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. Morbi euismod
        quam eget ex tincidunt dapibus. Donec vitae leo vehicula, fermentum urna
        vitae, gravida ex.
    </Text.Body>
    <Text.Body paragraph={true}>
        Aenean imperdiet faucibus velit, eu maximus libero facilisis ut. Donec
        nulla nisi, fermentum eget lorem at, feugiat ultricies ex. Aliquam
        volutpat nibh non suscipit rhoncus.
    </Text.Body>
</PreviewBox>

Comparing to if you did not specify it.

<PreviewBox>
    <Text.Body>
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. Morbi euismod
        quam eget ex tincidunt dapibus. Donec vitae leo vehicula, fermentum urna
        vitae, gravida ex.
    </Text.Body>
    <Text.Body>
        Aenean imperdiet faucibus velit, eu maximus libero facilisis ut. Donec
        nulla nisi, fermentum eget lorem at, feugiat ultricies ex. Aliquam
        volutpat nibh non suscipit rhoncus.
    </Text.Body>
</PreviewBox>

<Heading3>Switching underlying component</Heading3>

In some cases, you might require to change the underlying component to be
rendered. You may do so using the `as` prop.

> E.g. `Text.Body` renders a `<p>` but if you can specify another element to be
> rendered instead. Inspect the element to find out

<PreviewBox>
    <Text.Body>This is the original</Text.Body>
    <br />
    <Text.Body as="span">This becomes a span under the hood</Text.Body>
</PreviewBox>
