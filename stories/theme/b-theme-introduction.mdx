# Themes

<DocInfo>
    To use themes, you will need [**Styled
    Components**](https://styled-components.com/docs)
</DocInfo>

A theme specification is composed of several schemes, each representing a set of design tokens:

```tsx
// ThemeSpec
{
    colourScheme: "lifesg",
    fontScheme: "default",
    motionScheme: "default",
    borderScheme: "default",
    spacingScheme: "default",
    radiusScheme: "default",
    breakpointScheme: "default",
    resourceScheme: "lifesg"
}
```

Under the hood, a token is mapped to an actual value based on the theme.

## Usage

There are several presets available.

```tsx
// The LifeSG theme
import { LifeSGTheme } from "@lifesg/react-design-system/theme";

// The BookingSG theme
import { BookingSGTheme } from "@lifesg/react-design-system/theme";

// The CCUBE theme
import { CCubeTheme } from "@lifesg/react-design-system/theme";

// The MyLegacy theme
import { MyLegacyTheme } from "@lifesg/react-design-system/theme";

// The RBS theme
import { RBSTheme } from "@lifesg/react-design-system/theme";

// The OneService theme
import { OneServiceTheme } from "@lifesg/react-design-system/theme";

// The PA theme
import { PATheme } from "@lifesg/react-design-system/theme";

// The A11y Playground theme
import { A11yPlaygroundTheme } from "@lifesg/react-design-system/theme";

// The SupportGoWhere theme
import { SupportGoWhereTheme } from "@lifesg/react-design-system/theme";
```

To configure the theme in your project,

1. Import the theme preset
2. Wrap your components in the `ThemeProvider` context; typically this is done at the root of the application
3. Pass in the theme object via the `theme` prop

<DocNote>
    If a theme is not specified, components will use LifeSG as the default.
</DocNote>

```tsx
import { LifeSGTheme } from "@lifesg/react-design-system/theme";
import { ThemeProvider } from "styled-components";
import { Component } from "./index";

const App = () => {
    return (
        <ThemeProvider theme={LifeSGTheme}>
            <Component />
        </ThemeProvider>
    );
};

export default App;
```
