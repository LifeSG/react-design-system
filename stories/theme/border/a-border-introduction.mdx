# Border

```tsx
import { Border } from "@lifesg/react-design-system/theme";
```

The **border** tokens define the style and width of element borders. Utilities
are provided to declare certain border styles.

## Collections

[Default](/docs/foundations-border-default--docs)

## Usage

Apply the tokens when defining borders.

```tsx
import { Border } from "@lifesg/react-design-system/theme";

const container = css`
    border-width: ${Border["width-010"]};
    border-style: ${Border.solid};
`;
```

## Overriding

Refer to [this section](/docs/foundations-advanced--docs#theme-customisation) to
find out how to apply theme customisation.

Even though you will rarely need to override the border style properties, the
ability to do so is supported.

```tsx
useApplyStyle(themeElementRef, {
    [Border["width-010"]]: "6px", // specify a custom length value
    [Border["solid"]]: "dotted", // specify a custom border style
});
```
