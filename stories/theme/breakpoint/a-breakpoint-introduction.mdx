# Breakpoint

```tsx
import { Breakpoint, MediaQuery } from "@lifesg/react-design-system/theme";
```

Build responsive content across different screen widths using the **breakpoint**
and **media query** tokens. For grid-based page layouts, **column number** and
**column gutter** tokens are also available.

<DocInfo storybookPath="/docs/core-layout--docs">
    The `Layout` components let you compose page layouts based on the design
    tokens
</DocInfo>

## Usage

Use `MediaQuery` to target styles based on max width (desktop-first approach) or
min width (mobile-first approach).

### Max width

<DocAlert>
    Max-width media queries should be arranged in{" "}
    <strong>ascending order</strong>. The order matters as breakpoint detection
    is achieved with classes instead of native media queries.
</DocAlert>

```tsx
import { MediaQuery } from "@lifesg/react-design-system/theme";

const container = css`
    ${MediaQuery.MaxWidth.sm} {
        // styles
    }

    ${MediaQuery.MaxWidth.md} {
        // styles
    }

    ${MediaQuery.MaxWidth.lg} {
        // styles
    }
`;
```

### Min width

<DocAlert>
    Min-width media queries should be arranged in{" "}
    <strong>descending order</strong>. The order matters as breakpoint detection
    is achieved with classes instead of native media queries.
</DocAlert>

```tsx
import { MediaQuery } from "@lifesg/react-design-system/theme";

const container = css`
    ${MediaQuery.MaxWidth.lg} {
        // styles
    }

    ${MediaQuery.MaxWidth.md} {
        // styles
    }

    ${MediaQuery.MaxWidth.sm} {
        // styles
    }
`;
```

### Width as constant

```tsx
import {
    Breakpoint,
    parsePxOrRemValue,
    useDesignToken,
} from "@lifesg/react-design-system/theme";

export const MyComponent = () => {
    const theme = useTheme();
    // TODO: fix this example
    const tablet = parsePxOrRemValue(useDesignToken(Breakpoint["md-min"]));

    if (window.innerWidth > tablet) {
        // render something
    } else {
        // render something else
    }
};
```

## Overriding

Refer to [this section](/docs/foundations-advanced--docs#theme-customisation) to
find out how to apply theme customisation.

```tsx
useApplyStyle(themeElementRef, {
    [Breakpoint["sm-max"]]: "430px", // specify a custom length value
    [Breakpoint["md-min"]]: "431px", // specify a custom length value
    [Breakpoint["xxs-column"]]: 4, // specify a custom integer value
    [Breakpoint["xxs-gutter"]]: "8px", // specify a custom length value
    [Breakpoint["xxs-margin"]]: "20px", // specify a custom length value
});
```
