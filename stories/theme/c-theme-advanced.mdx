# Advanced usage

## Theme customisation

### In CSS

You can simply override a design token in your stylesheet like any other CSS
variable.

```css
[data-fds-theme="lifesg"] {
    --fds-colour-bg-primary: #fab;
}

[data-fds-theme="lifesg"][data-fds-theme-mode="dark"] {
    --fds-colour-bg-primary: #fab;
}
```

### In JavaScript

If you need to dynamically change the theme, you can override CSS variables
programmatically through the inline `style` prop.

```tsx
import { ThemeProvider } from "@lifesg/react-design-system/theme";

const App = () => {
    return (
        <ThemeProvider
            style={{
                "--fds-colour-bg-primary": "#fab",
            }}
        />
    );
};
```

### In JavaScript (strict CSP)

If your project has a strict CSP configuration and cannot support inline styles,
you can use the `useApplyStyle` hook.

```tsx
import { ThemeProvider } from "@lifesg/react-design-system/theme";

const App = () => {
    const themeElementRef = useRef<HTMLDivElement>(null);

    useApplyStyle(themeElementRef, {
        "--fds-colour-bg-primary": "#fab",
    });

    return <ThemeProvider ref={themeElementRef} />;
};
```

## Nested themes

If you need to selectively apply another theme to a different part of your
application, that can be achieved by nesting `ThemeProvider`s.

```tsx
import { ThemeProvider } from "@lifesg/react-design-system/theme";

const App = (
    <ThemeProvider theme="lifesg">
        <Button />
        <ThemeProvider theme="bookingsg">
            <Button />
        </ThemeProvider>
    </ThemeProvider>
);
```

## Portal support

For components rendered in a portal, they typically sit outside of your
application's main HTML DOM tree and will not inherit the theme. A
`useInheritedThemeVariables` hook is provided to apply the theme to the portal
root element.

```tsx
const PortalledComponent = () => {
    const [show, setShow] = useState(false);
    const { themeProps, themeStyle } = useInheritedThemeScope(show);

    return show
        ? createPortal(
              <div {...themeProps} style={{ ...themeStyle }}>
                  {/* Actual content here */}
              </div>,
              document.body
          )
        : null;
};
```
