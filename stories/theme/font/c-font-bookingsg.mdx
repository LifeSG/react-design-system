# BookingSG font

This is the font set used when the `fontScheme` is `"bookingsg"`.

```tsx
const theme: ThemeSpec = {
    fontScheme: "bookingsg",
    // ...other specifications
};
```

This stylesheet should be loaded in your app.

```
https://assets.life.gov.sg/react-design-system/v3/css/plus-jakarta-sans.css
```

Note that the _font ligatures_ is **_disabled_** by default for this `fontScheme`.
