# PA font

This is the font set used when the `fontScheme` is `"pa"`.

```tsx
const theme: ThemeSpec = {
    fontScheme: "pa",
    // ...other specifications
};
```

This stylesheet should be loaded in your app.

```
https://assets.life.gov.sg/react-design-system/v3/css/lato.css
```
