# A11y Playground font

This is the font set used when the `fontScheme` is `"a11y"`.

```tsx
const theme: ThemeSpec = {
    fontScheme: "a11y",
    // ...other specifications
};
```

This stylesheet should be loaded in your app.

```
https://assets.life.gov.sg/react-design-system/v3/css/atkinson-hyperlegible-next.css
```
