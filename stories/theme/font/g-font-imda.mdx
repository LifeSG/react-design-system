# SupportGoWhere font

This is the font set used when the `fontScheme` is `"imda"`.

```tsx
const theme: ThemeSpec = {
    fontScheme: "imda",
    // ...other specifications
};
```

This stylesheet should be loaded in your app.

```
https://assets.life.gov.sg/react-design-system/v3/css/montserrat.css
```
