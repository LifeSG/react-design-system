# SGW Digital Lobby font

This is the font set used when the `fontScheme` is `"sgw-digital-lobby"`.

```tsx
const theme: ThemeSpec = {
    fontScheme: "sgw-digital-lobby",
    // ...other specifications
};
```

This stylesheet should be loaded in your app.

```
https://assets.life.gov.sg/react-design-system/v3/css/libre-franklin.css
```
