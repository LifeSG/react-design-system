# SPF font

This is the font set used when the `fontScheme` is `"spf"`.

```tsx
const theme: ThemeSpec = {
    fontScheme: "spf",
    // ...other specifications
};
```

This stylesheet should be loaded in your app.

```
https://assets.life.gov.sg/react-design-system/v3/css/public-sans.css
```
