# SMGS font

This is the font set used when the `fontScheme` is `"smgs"`.

```tsx
const theme: ThemeSpec = {
    fontScheme: "smgs",
    // ...other specifications
};
```

These stylesheets should be loaded in your app.

```
// Poppins
https://assets.life.gov.sg/react-design-system/v3/css/poppins.css

// Inter
https://assets.life.gov.sg/react-design-system/v3/css/inter.css
```

<DocInfo>
    [Common
    ligatures](https://developer.mozilla.org/en-US/docs/Web/CSS/font-variant-ligatures)
    are **disabled** for this font set
</DocInfo>
