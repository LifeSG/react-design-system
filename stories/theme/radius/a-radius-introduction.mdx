# Radius

```tsx
import { Radius } from "@lifesg/react-design-system/theme";
```

The **radius** tokens define the border radius of elements.

## Collections

[Default](/docs/foundations-radius-default--docs)

## Usage

Apply the tokens when defining border radius.

```tsx
import { Radius } from "@lifesg/react-design-system/theme";

const container = css`
    border-radius: ${Radius["sm"]};
`;
```

## Overriding

Refer to [this section](/docs/foundations-advanced--docs#theme-customisation) to
find out how to apply theme customisation.

```tsx
useApplyStyle(themeElementRef, {
    [Radius["sm"]]: "6px", // specify a custom length value
});
```
