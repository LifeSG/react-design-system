# Radius

```tsx
import { Radius } from "@lifesg/react-design-system/theme";
```

The **radius** tokens define the border radius of elements.

## Collections

[LifeSG](/docs/foundations-radius-lifesg--docs)

## Usage

Apply the tokens when defining border radius.

```tsx
import { Radius } from "@lifesg/react-design-system/theme";
import styled from "styled-components";

const Container = styled.div`
    border-radius: ${Radius["sm"]};
`;
```

## Overriding

```tsx
import { LifeSGTheme } from "@lifesg/react-design-system/theme";

const customTheme: ThemeSpec = {
    ...LifeSGTheme,
    overrides: {
        radius: {
            sm: 6, // specify a custom pixel-based value
        },
    },
};
```
