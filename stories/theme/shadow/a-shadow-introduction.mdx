# Shadow

```tsx
import { Shadow } from "@lifesg/react-design-system/theme";
```

The **shadow** tokens define the drop shadow of elements and imply elevation.

## Collections

[Default](/docs/foundations-shadow-default--docs)

## Usage

Apply the tokens when defining box shadows.

```tsx
import { Shadow } from "@lifesg/react-design-system/theme";

const container = css`
    box-shadow: ${Shadow["sm-subtle"]};
`;
```

## Overriding

Refer to [this section](/docs/foundations-advanced--docs#theme-customisation) to
find out how to apply theme customisation.

```tsx
useApplyStyle(themeElementRef, {
    [Shadow["sm-subtle"]]: "0 0 4px 1px rgba(0, 0, 0, 0.5)", // specify a custom box shadow value
});
```
