<Title>TimeSlotBarWeek</Title>

<Secondary>Overview</Secondary>

Displays a calendar component to view available time slots in a week.

```tsx
import { TimeSlotBarWeek } from "@lifesg/react-design-system/time-slot-bar-week";
```

<Canvas of={TimeSlotBarWeekStories.Default} />

<Heading3>Interval</Heading3>

You can use `interval` to control the duration (minutes) each cell represents.

<Canvas of={TimeSlotBarWeekStories.Interval} />

<Heading3>Variations</Heading3>

The component also comes in a `fixed` variant (`flexible` by default).

<Canvas of={TimeSlotBarWeekStories.Variations} />

<Heading3>Max visible cell height</Heading3>

You can use `maxVisibleCellHeight` to set the maximum height (pixels) for the combined cells within the component, before truncation occurs.

<Canvas of={TimeSlotBarWeekStories.MaxVisibleCellHeight} />

<Heading3>Start and end time range</Heading3>

You can specify the `startTime` and `endTime` to to set the minimum and maximum time range displayed.

_(If unspecified, `startTime` and `endTime` will be determined based on earliest/latest time slots across all dates)_

<Canvas of={TimeSlotBarWeekStories.StartAndEndTime} />

<Heading3>Within minimum and maximum dates</Heading3>

You can specify the `minDate` and/or `maxDate` to control the selection range.

<Canvas of={TimeSlotBarWeekStories.MinAndMaxDates} />

<Heading3>With disabled dates</Heading3>

You can use `disabledDates` to specify dates that are not available for selection.

_(In this example, 2 days before today and 2 days after today are disabled)_

<Canvas of={TimeSlotBarWeekStories.WithDisabledDates} />

<Heading3>Hiding nav header</Heading3>

You can remove the navigation header, including the month, year dropdown.

<Canvas of={TimeSlotBarWeekStories.HideNavHeader} />

<Heading3>Disable day selection</Heading3>

<Canvas of={TimeSlotBarWeekStories.DisableDaySelection} />

<Heading3>Handle slot selection</Heading3>

<Canvas of={TimeSlotBarWeekStories.SlotSelection} />
