<Title>TimeSlotBarWeek</Title>

<Secondary>Overview</Secondary>

Displays a calendar component to view available time slots in a week.

```tsx
import { TimeSlotBarWeek } from "@lifesg/react-design-system/time-slot-bar-week";
```

<Canvas>
    <Story name="TimeSlotBarWeek">
        <Wrapper>
            <TimeSlotBarWeek slots={getSlots()} />
        </Wrapper>
    </Story>
</Canvas>

<Heading3>Interval</Heading3>

You can use `interval` to control the duration (minutes) each cell represents.

<Canvas>
    <Story name="Interval">
        {() => {
            return (
                <Wrapper>
                    <TimeSlotBarWeek slots={getSlots()} interval={15} />
                </Wrapper>
            );
        }}
    </Story>
</Canvas>

<Heading3>Variations</Heading3>

The component also comes in a `fixed` variant (`flexible` by default).

<Canvas>
    <Story name="Variations">
        {() => {
            return (
                <Wrapper>
                    <TimeSlotBarWeek
                        slots={getSlots()}
                        interval={15}
                        variant={"fixed"}
                    />
                </Wrapper>
            );
        }}
    </Story>
</Canvas>

<Heading3>Truncated height</Heading3>

You can use `truncatedHeight` to set the maximum height of the combined cells (pixels) before truncation.

<Canvas>
    <Story name="Truncated height">
        {() => {
            return (
                <Wrapper>
                    <TimeSlotBarWeek slots={getSlots()} truncatedHeight={136} />
                </Wrapper>
            );
        }}
    </Story>
</Canvas>

<Heading3>Start and end time range</Heading3>

You can specify the `startTime` and `endTime` to to set the minimum and maximum time range displayed.

_(If unspecified, `startTime` and `endTime` will be determined based on earliest/latest time slots across all dates)_

<Canvas>
    <Story name="Min and max time">
        {() => {
            return (
                <Wrapper>
                    <TimeSlotBarWeek
                        slots={getSlots()}
                        startTime="10:30"
                        endTime="15:30"
                    />
                </Wrapper>
            );
        }}
    </Story>
</Canvas>

<Heading3>Within minimum and maximum dates</Heading3>

You can specify the `minDate` and/or `maxDate` to control the selection range.

<Canvas>
    <Story name="Min and max dates">
        {() => {
            const minDate = dayjs().subtract(7, "day").format("YYYY-MM-DD");
            const maxDate = dayjs().add(20, "day").format("YYYY-MM-DD");
            return (
                <Wrapper>
                    <TimeSlotBarWeek
                        slots={getSlots()}
                        minDate={minDate}
                        maxDate={maxDate}
                    />
                </Wrapper>
            );
        }}
    </Story>
</Canvas>

<Heading3>With disabled dates</Heading3>

You can use `disabledDates` to specify dates that are not available for selection.

_(In this example, 2 days before today and 2 days after today are disabled)_

<Canvas>
    <Story name="With disabled dates">
        {() => {
            const [disabledDates] = useState([
                dayjs().subtract(2, "day").format("YYYY-MM-DD"),
                dayjs().add(2, "day").format("YYYY-MM-DD"),
            ]);
            return (
                <Wrapper>
                    <TimeSlotBarWeek
                        slots={getSlots()}
                        disabledDates={disabledDates}
                    />
                </Wrapper>
            );
        }}
    </Story>
</Canvas>

<Heading3>Hiding nav header</Heading3>

You can remove the navigation header, including the month, year dropdown.

<Canvas>
    <Story name="Hide nav header">
        {() => {
            return (
                <FullWidthStoryContainer>
                    <Wrapper>
                        <TimeSlotBarWeek
                            slots={getSlotsMin()}
                            showNavigationHeader={false}
                        />
                    </Wrapper>
                </FullWidthStoryContainer>
            );
        }}
    </Story>
</Canvas>

<Heading3>Disable day selection</Heading3>

<Canvas>
    <Story name="Disable day selection">
        <FullWidthStoryContainer>
            <Wrapper>
                <TimeSlotBarWeek slots={getSlots()} enableSelection={false} />
            </Wrapper>
        </FullWidthStoryContainer>
    </Story>
</Canvas>

<Heading3>Handle slot selection</Heading3>
