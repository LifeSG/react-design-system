# TimeSlotBar

## Overview

A component that displays time slots for a specified time range.

```tsx
import { TimeSlotBar } from "@lifesg/react-design-system/time-slot-bar";
```

<Canvas of={TimeSlotBarStories.Default} />

## Minified Variant

In cases where there is a requirement to display a large number of time slots in a smaller viewport, you can make use of the minified variant of the component. To do so, set `variant="minified"`.
