# Timeline

## Overview

A component that displays upcoming steps of a process in a timeline presentation.

```tsx
import { Timeline } from "@lifesg/react-design-system/timeline";
```

There are a variety of contents available to be specified for each `Timeline` item.

<Canvas of={TimelineStories.Default} />

## Variations

The `Timeline` component comes in 4 variants and each item can be styled individually.
