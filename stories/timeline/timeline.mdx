# Timeline

## Overview

A component that displays upcoming steps of a process in a timeline presentation.

```tsx
import { Timeline } from "@lifesg/react-design-system/timeline";
```

There are a variety of contents available to be specified for each `Timeline` item.

<Canvas of={TimelineStories.Default} />

## Variations

The `Timeline` component comes in 5 variants and each item can be styled individually.

<Canvas of={TimelineStories.Variants} />

## Numeric variation

The `numeric` variant displays a numbered index for each item, useful for showing sequential steps in a process.

<Canvas of={TimelineStories.Numeric} />

### Numeric with offset

The `counterOffset` prop allows you to start the numeric indicators at a specific number. For example, with `counterOffset={2}`, the numbering starts at 3 (following the formula: `index + 1 + offset`). This is useful when displaying a subset of steps from a larger process.

<Canvas of={TimelineStories.NumericWithOffset} />
