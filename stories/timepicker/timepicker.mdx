<Title>Timepicker</Title>

<Secondary>Overview</Secondary>

A text input that opens a time-selection overlay for picking hours and minutes. Supports both 12-hour (AM/PM) and 24-hour display formats. Controlled via a formatted time string.

```tsx
import { Timepicker } from "@lifesg/react-design-system/timepicker";
```
