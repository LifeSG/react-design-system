# Toggle

## Overview

A button that allows a user to enter a response by clicking, and shown with a different state when activated.

```tsx
import { Toggle } from "@lifesg/react-design-system/toggle";
```

<Canvas of={ToggleStories.Default} />

## Behaviours

The `Toggle` can be used as a checkbox or radio button. This can be determined by the `type` prop.

| Type       | Behaviour                         |
| ---------- | --------------------------------- |
| `checkbox` | Checkbox, deselection is allowed  |
| `radio`    | Radio, deselection is not allowed |
| `yes`/`no` | Radio, deselection is not allowed |
