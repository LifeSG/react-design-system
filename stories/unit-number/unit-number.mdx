A structured input for Singapore-style unit numbers with separate floor and unit fields (e.g., `#12-345`). Supports controlled and uncontrolled modes with raw and formatted value callbacks.

```tsx
import { UnitNumberInput } from "@lifesg/react-design-system/unit-number";
```
