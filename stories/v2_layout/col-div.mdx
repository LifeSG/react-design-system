<Title>Column Divs</Title>

<Secondary>Overview</Secondary>

A div that conforms to the grid layout of `Layout.Container`.

```tsx
import { V2_Layout } from "@lifesg/react-design-system/v2_layout";
```

<Heading3>Anatomy</Heading3>

The Column Divs allows specifying the column widths for each viewport pertaining
to the specifications of the [grid system](/docs/getting-started-layout--docs#grid-layout).
**By default, the component will only take up 1 column in the grid regardless of
the viewport.**

There are 3 levels of viewport specific props that you can set. Here is a rule
table to denote the priority of how the viewport specific props are applied.

| Viewport | `mobileCols` | `tabletCols` | `desktopCols` | To use        |
| -------- | ------------ | ------------ | ------------- | ------------- |
| Desktop  | Defined      | -            | -             | `mobileCols`  |
|          | Defined      | Defined      | -             | `tabletCols`  |
|          | Defined      | Defined      | Defined       | `desktopCols` |
|          | Defined      | -            | Defined       | `desktopCols` |
|          | -            | -            | Defined       | `desktopCols` |
| Tablet   | Defined      | -            | -             | `mobileCols`  |
|          | Defined      | Defined      | -             | `tabletCols`  |
|          | Defined      | Defined      | Defined       | `tabletCols`  |
|          | Defined      | -            | Defined       | `mobileCols`  |
|          | -            | -            | Defined       | Default 1 col |
| Mobile   | Defined      | -            | -             | `mobileCols`  |
|          | Defined      | Defined      | -             | `mobileCols`  |
|          | Defined      | Defined      | Defined       | `mobileCols`  |
|          | Defined      | -            | Defined       | `mobileCols`  |
|          | -            | -            | Defined       | Default 1 col |
|          | -            | Defined      | -             | Default 1 col |
