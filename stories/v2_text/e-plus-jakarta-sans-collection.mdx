# Plus Jakarta Sans Style

## Overview

This will be the collection used when the `textStyleScheme` is `plusJakartaSans`.

```tsx
const theme: ThemeSpec = {
    textStyleScheme: "plusJakartaSans",
    // ...other specifications
};
```
